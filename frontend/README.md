# Frontend — Alerte École

App Expo / React Native (web + iOS + Android), TypeScript.

👉 **La documentation complète (architecture, lancement, comptes de démo, API) est dans le
[README principal](../README.md) à la racine du projet.**

## Démarrage rapide

```bash
npm install
npm run web      # version web
# ou
npm start        # puis 'i' (iOS), 'a' (Android), ou scanner le QR avec Expo Go
```

⚠️ Le backend doit tourner (`backend/`, port 8800) et MySQL être lancé (`docker compose up -d`
depuis la racine). Voir le [README principal](../README.md).
